Meet the Team

Paige Cherry

Role: Data Modeling Specialist

Investigating giant tortoise ecological radiations using advanced methods under Dr. Roman. Leveraging R/Python, web design, and communication skills to analyze and present research findings effectively.

Sara Cielaszyk

Role: Research and Visualization Lead

Lead data analyst, explored a connection between body size and island size, ran correlation and made interactive map to show these findings. Estimated the diversification rates for the phylogenic trees using MiSSE.

Hubert Kasprzycki

Role: Backend Web Developer

Conceptualized, designed, and implemented user-friendly, interactive webpages for the website.Worked on the backend of the websites to incorporate the data on the page and tackled bugs.

Eleanor Tuck

Role: Frontend Web Developer

Developed the core structure of the project website using HTML and organized its overall content layout. Designed interactive plots with Plotly and integrated image features to visually communicate key findings and enhance user experience.
